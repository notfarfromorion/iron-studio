ZStreamdeflateParams Method

Dynamically update the compression level and compression strategy. The interpretation of level is as in deflateInit(Int32). This can be used to switch between compression and straight copy of the input data, or to switch to a different kind of input data requiring a different strategy. If the compression level is changed, the input available so far is compressed with the old level (and may be flushed); the new level will take effect only at the next call of deflate(FlushStrategy)

Return Value
Type: Int32

            deflateParams returns Z_OK if success, Z_STREAM_ERROR if the source stream 
            state was inconsistent or if a parameter was invalid, Z_BUF_ERROR if avail_out was zero.
            
Remarks

            Before the call of deflateParams(Int32, CompressionStrategy), the stream state must be set as for a call of deflate(FlushStrategy), since the 
            currently available input may have to be compressed and flushed. In particular, avail_out must be non-zero.
