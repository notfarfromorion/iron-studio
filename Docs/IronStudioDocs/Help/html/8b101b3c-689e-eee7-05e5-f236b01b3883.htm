ClrNameOpsMangle Method

Converts a camelCase or PascalCase name to a Ruby name (e.g. "FooBar" to "foo_bar"). Returns null if the name is not in camelCase or PascalCase (FooBAR, foo, etc.). Characters that are not upper case letters are treated as lower-case letters.
