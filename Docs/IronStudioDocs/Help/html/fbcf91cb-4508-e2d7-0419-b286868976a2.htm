ObjectOperationsTryConvertTo Method (ObjectHandle, Type, ObjectHandle)

Converts the remote object into the specified type returning a handle to the new remote object. Returns true if the value can be converted, false if it cannot. The conversion will be explicit or implicit depending on what the langauge prefers.
