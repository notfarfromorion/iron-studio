PythonCompilerOptionsInitialIndent Property

Gets or sets the initial indentation. This can be set to allow parsing partial blocks of code that are already indented. For each element of the array there is an additional level of indentation. Each integer value represents the number of spaces used for the indentation. If this value is null then no indentation level is specified.
