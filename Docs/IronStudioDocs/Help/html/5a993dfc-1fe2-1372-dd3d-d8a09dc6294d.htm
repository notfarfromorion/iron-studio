See PEP 342 (http://python.org/dev/peps/pep-0342/) for details of new methods on Generator. Full signature including default params for throw is: throw(type, value=None, traceback=None) Use multiple overloads to resolve the default parameters.
throw(Object, Object)
throw(Object, Object, Object)
Throw(...) is like Raise(...) being called from the yield point within the generator. Note it must come from inside the generator so that the traceback matches, and so that it can properly cooperate with any try/catch/finallys inside the generator body. If the generator catches the exception and yields another value, that is the return value of g.throw().
