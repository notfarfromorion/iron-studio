Expression Class

Represents expressions. Statements are considered special cases of expressions in AST class hierarchy. Unlike syntactic expression a syntactic statement cannot be assigned to a left value. However certain Ruby constructs (e.g. block-expression) allow to read the value of a statement. Usually such value is null (e.g. undef, alias, while/until statements), although some syntactic statements evaluate to a non-null value (e.g. if/unless-statements).
