BindingTarget Properties

The BindingTarget type exposes the following members.

Properties
  NameDescription
ActualArgumentCount
Returns the total number of arguments provided to the call. 0 if the call succeeded or failed for a reason other than argument count mismatch.
AmbiguousMatches
Returns the methods which don't have any matches or null if Result == BindingResult.AmbiguousMatch
CallFailures
Returns the methods and their associated conversion failures if Result == BindingResult.CallFailure.
ExpectedArgumentCount
Returns the acceptable number of arguments which can be passed to the method if Result == BindingResult.IncorrectArgumentCount.
Method Obsolete.
Returns the method if the binding succeeded, or null if no method was applicable.
MethodCandidate
Returns the MethodTarget if the binding succeeded, or null if no method was applicable.
Name
Gets the name of the method as supplied to the OverloadResolver.
NarrowingLevel
Returns the NarrowingLevel of the method if the call succeeded. If the call failed returns NarrowingLevel.None.
Overload
Returns the selected overload if the binding succeeded, or null if no one was applicable.
RestrictedArguments
Gets the MetaObjects which we originally did binding against in their restricted form. The members of the array correspond to each of the arguments. All members of the array have a value.
Result
Gets the result of the attempt to bind.
ReturnType
Returns the return type of the binding, or null if no method was applicable.
Success
Returns true if the binding was succesful, false if it failed. This is an alias for BindingTarget.Result == BindingResult.Success.
