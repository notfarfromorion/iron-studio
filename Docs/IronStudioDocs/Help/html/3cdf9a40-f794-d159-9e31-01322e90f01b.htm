FunctionCode Properties

The FunctionCode type exposes the following members.

Properties
  NameDescription
co_argcount
co_cellvars
Returns a list of variable names which are accessed from nested functions.
co_code
Returns the byte code. IronPython does not implement this and always returns an empty string for byte code.
co_consts
Returns a list of constants used by the function. The first constant is the doc string, or None if no doc string is provided. IronPython currently does not include any other constants than the doc string.
co_filename
Returns the filename that the code object was defined in.
co_firstlineno
Returns the 1st line number of the code object.
co_flags
Returns a set of flags for the function. 0x04 is set if the function used *args 0x08 is set if the function used **args 0x20 is set if the function is a generator
co_freevars
Returns a list of free variables (variables accessed from an outer scope). This does not include variables accessed in the global scope.
co_lnotab
Returns a mapping between byte code and line numbers. IronPython does not implement this because byte code is not available.
co_name
Returns the name of the code (function name, class name, or <module>).
co_names
Returns a list of global variable names accessed by the code.
co_nlocals
Returns the number of local varaibles defined in the function.
co_stacksize
Returns the stack size. IronPython does not implement this because byte code is not supported.
