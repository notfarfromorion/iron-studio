ScopeGetVariable(Scope, String)
Provides access to getting variables in scopes. By default this goes through ObjectOperations which can be rather slow. Languages can override this to provide fast customized access which avoids ObjectOperations. Languages can provide fast access to commonly used scope types for that language. Typically this includes ScopeStorage and any other classes which the language themselves uses for backing of a Scope.
