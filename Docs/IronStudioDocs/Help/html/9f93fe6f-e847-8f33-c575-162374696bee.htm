Equal(BigInteger, BigInteger)
Returns true if other has the same value as self, where other is Fixnum or Bignum. Contrast this with Bignum#eql?, which requires other to be a Bignum.
Equal(BigInteger, Int32)
Returns true if other has the same value as self, where other is Fixnum or Bignum. Contrast this with Bignum#eql?, which requires other to be a Bignum.
Equal(BinaryOpStorage, BigInteger, Object)
Returns true if other has the same value as self, where other is not Fixnum, Bignum or Float. Contrast this with Bignum#eql?, which requires other to be a Bignum.
Equal(RubyContext, BigInteger, Double)
Returns true if other has the same value as self, where other is Float. Contrast this with Bignum#eql?, which requires other to be a Bignum.
