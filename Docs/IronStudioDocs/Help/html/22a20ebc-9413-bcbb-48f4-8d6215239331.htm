WpfLoadComponent Method (CodeContext, Object, TextReader)

Loads XAML from the specified XmlReader and returns the deserialized object. Any event handlers are bound to methods defined in the provided module. Any named objects are assigned to the object. The provided object is expected to be the same type as the root of the XAML element.
