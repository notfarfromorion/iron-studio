TextContentProvider Class

Provides a factory to create TextReader's over one source of textual content. TextContentProvider's are used when reading from a source which is already decoded or has a known specific decoding. For example a text editor might provide a TextContentProvider whose backing is an in-memory text buffer that the user can actively edit.
