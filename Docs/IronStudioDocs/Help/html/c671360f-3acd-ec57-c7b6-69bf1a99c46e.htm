ScriptingRuntimeHelpers Class

These are some generally useful helper methods. Currently the only methods are those to cached boxed representations of commonly used primitive types so that they can be shared. This is useful to most dynamic languages that use object as a universal type. The methods in RuntimeHelepers are caleld by the generated code. From here the methods may dispatch to other parts of the runtime to get bulk of the work done, but the entry points should be here.
