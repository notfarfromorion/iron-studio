CreateScriptSource(CodeObject)
This method returns a ScriptSource object from a System.CodeDom.CodeObject. This is a factory method for creating a ScriptSources with this language binding. The expected CodeDom support is extremely minimal for syntax-independent expression of semantics. Languages may do more, but hosts should only expect CodeMemberMethod support, and only sub nodes consisting of the following: CodeSnippetStatement CodeSnippetExpression CodePrimitiveExpression CodeMethodInvokeExpression CodeExpressionStatement (for holding MethodInvoke)
