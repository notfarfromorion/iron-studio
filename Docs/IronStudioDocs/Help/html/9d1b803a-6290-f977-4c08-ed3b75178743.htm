BuiltinPythonModulePerformModuleReload Method

Called when the user attempts to reload() on your module and by the base class Initialize method. This provides an opportunity to allocate any per-module data which is not simply function definitions. A common usage here is to create exception objects which are allocated by the module using PythonExceptions.CreateSubType.
