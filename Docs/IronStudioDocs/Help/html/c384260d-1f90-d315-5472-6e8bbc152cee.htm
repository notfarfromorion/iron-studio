Methods

GetExpressions
Returns the list of expressions represented by the DynamicMetaObject instances.
LightBindT
Produces an interpreted binding using the given binder which falls over to a compiled binding after hitCount tries. This method should be called whenever an interpreted binding is required. Sometimes it will return a compiled binding if a previous binding was produced and it's hit count was exhausted. In this case the binder will not be called back for a new binding - the previous one will be used.
ObjectToMetaObject
Creates an instance of DynamicMetaObject for a runtime value and the expression that represents it during the binding process.
