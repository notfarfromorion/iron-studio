IWeakReferenceableSetFinalizer Method

Sets a WeakRefTracker on an object for the purposes of supporting finalization. All user types (new-style and old-style) support finalization even if they don't support weak-references, and therefore this function always succeeds. Note the slot used to store the WeakRefTracker is still shared between SetWeakRef and SetFinalizer if a type supports both.
