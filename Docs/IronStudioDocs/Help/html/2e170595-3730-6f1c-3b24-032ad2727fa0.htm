LibraryManagerOnNewFile Method

Overridden in the base class to receive notifications of when a file should be analyzed for inclusion in the library. The derived class should queue the parsing of the file and when it's complete it should call FileParsed with the provided LibraryTask and an IScopeNode which provides information about the members of the file.
